# Jogo-Da-Velha-AOC

# Projeto desenvolvido para a disciplina de Infraestrutura de Hardware, implementando um Jogo da Velha em Assembly MIPS, totalmente funcional no simulador MARS.

# O programa utiliza sub-rotinas para organizar a lógica do jogo, manipulação do tabuleiro, verificação de vitória/empate e interação com o usuário (Player vs CPU).
